1. Install XCode before install brew
```sh
xcode-select --install
```

2. Install Java -- after homebrew core installed

```sh
brew install java

# make link
sudo ln -sfn /usr/local/opt/openjdk/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk.jdk
```
